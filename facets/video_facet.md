# VideoFacet type
VideoFacet groups video related data into a single structure.
It is available on the video property of [Item](item.md) resources that have associated video.

### JSON representation
(ODB-Support: coming soon)
```json
{
	"bitrate": {float},
	"duration": {string},
	"height": {integer},
	"width": {integer},
	"fourCC": {string}
}
```
